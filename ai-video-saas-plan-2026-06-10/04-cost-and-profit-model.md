# 成本与净利润模型

## 核心原则

产品不能按「成片分钟数」直接等同「AI 视频生成分钟数」。

正确结构：

```text
1 分钟成片 =
40-50 秒低成本图文 / 口播 / 字幕 / 素材混剪
+ 5-15 秒 AI 高级镜头
+ 1080p30 后处理
```

这样才能把成本压住。

## 单条视频成本

| 视频类型 | 内容结构 | 内部成本/1 分钟 | 是否适合普通套餐 |
|---|---|---:|---|
| 轻量图文视频 | 商品图 / 店铺图 + TTS + 字幕 + 模板动效 | ¥0.5-5 | 是 |
| 标准小店视频 | 轻视频 + 数字人口播 / 5-15 秒 WAN 镜头 | ¥8-40 | 是，但限额 |
| 高级视频 | 大量 WAN / Seedance 生成镜头 | ¥60-300+ | 不适合无限送 |
| 全 AI 生成视频 | 整条视频都用视频模型生成 | ¥100-600+/分钟 | 只适合高级加购 |

## 套餐建议

不要卖「无限高级 AI 视频」。建议用订阅 + 点数。

| 套餐 | 建议价格 | 权益方向 | 备注 |
|---|---:|---|---|
| 免费试用 | ¥0 / ¥9.9 | 3-5 条带水印轻视频 | 用于获客 |
| 入门版 | ¥59/月 | 轻视频为主，少量 TTS | 不送或极少高级镜头 |
| 创作者版 | ¥129/月 | 轻视频 + 少量数字人 + 少量 WAN | 小博主 / 小店 |
| 小店版 | ¥199/月 | 店铺模板、商品模板、更多口播 | 本地商家 |
| 电商 Pro | ¥299/月 | 批量商品、更多点数、优先队列 | 主推 |
| 增长版 | ¥599/月 | 多账号、多店铺、更高点数 | 高频用户 |

## 净利润公式

```text
月收入
- 模型/API/云 GPU 成本
- 基础服务器/存储/带宽
- 支付手续费
- 退款/失败补偿
- 合规审核成本
- 法务/会计/税务成本
- 自媒体推流/获客成本
- 客服/运营/人工
- 增值税及附加
= 企业所得税前利润

- 企业所得税
= 公司税后净利润

如果分红给个人：
- 股息红利个人所得税
= 老板个人可拿到手的钱
```

## 三档测算

| 指标 | 早期验证 | 小规模可跑 | 初步成生意 |
|---|---:|---:|---:|
| 付费用户 | 100 | 500 | 2000 |
| ARPU | ¥129/月 | ¥159/月 | ¥179/月 |
| 月收入 | ¥12,900 | ¥79,500 | ¥358,000 |
| 模型/API/云 GPU + 基础云 | ¥6,000 | ¥33,000 | ¥145,000 |
| 支付手续费 + 退款补偿 | ¥335 | ¥2,067 | ¥9,300 |
| 合规/审核/法务/会计 | ¥3,300 | ¥10,000 | ¥45,000 |
| 自媒体推流/获客 | ¥2,000 | ¥8,000 | ¥40,000 |
| 客服/运营/人工 | ¥0 | ¥8,000 | ¥45,000 |
| 增值税及附加 | ¥0 | ¥0 | 约 ¥3,800 |
| 企业所得税 | 约 ¥63 | 约 ¥1,320 | 约 ¥3,500 |
| 公司税后净利润 | 约 ¥1,200 | 约 ¥25,000 | 约 ¥66,000 |

解释：

- 100 个付费用户：验证阶段，基本不养人。
- 500 个付费用户：公司开始有正现金流。
- 2000 个付费用户：开始像一门稳定 SaaS 生意。

## 健康指标

建议目标：

```text
ARPU >= ¥159
单用户生成成本 <= ARPU 的 30%-40%
高级模型成本 <= 总收入的 15%-25%
退款率 <= 3%
CAC <= ¥100-150
LTV >= CAC × 3
公司税后净利率 >= 15%-25%
```

## 不能踩的坑

- 不要无限送 Seedance / WAN。
- 不要把全片 AI 生成放进低价套餐。
- 不要忽略失败重抽。
- 不要只算模型成本，不算税、退款、客服、审核。
- 不要让境外 API 成本完全没有财税凭证。
- 不要把用户素材和真人肖像当普通图片随便处理。
