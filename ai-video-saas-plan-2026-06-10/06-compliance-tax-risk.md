# 合规、税务与风险

## 重要说明

本文件是商业和产品规划视角的合规清单，不构成正式法律或税务意见。上线前需要让会计、税务顾问、律师按主体所在地和真实业务模式确认。

## 必须纳入成本的项目

| 项目 | 一次性成本 | 月度成本 |
|---|---:|---:|
| 公司注册 / 基础财税 | ¥0-3000 | ¥500-1500 |
| 记账报税 | - | ¥500-2000 |
| 用户协议 / 隐私政策 / 付费协议 | ¥5000-30000 | 修改另算 |
| 肖像 / 声音 / 素材授权条款 | ¥3000-20000 | 修改另算 |
| AI 内容标识 / 水印 / 元数据 | ¥3000-20000 开发成本 | 很低 |
| 内容审核规则 / 敏感词 / 黑名单 | ¥3000-15000 | ¥1000-30000 |
| ICP 备案 / 公安备案 | 多数免费，代办 ¥0-2000 | 无 |
| ICP 许可证判断 / 咨询 | ¥3000-20000 | 无 |
| 数据出境 / 个人信息合规咨询 | ¥5000-50000 | 视规模 |
| 商标注册 | ¥300-1500/类 | 无 |
| 投诉 / 律师函 / 争议处理预留 | - | ¥1000-10000 |

MVP 建议预留：

```text
一次性合规启动：¥20,000-50,000
月度合规运营：¥3,000-10,000
```

正式商业化后建议预留：

```text
一次性合规建设：¥50,000-150,000
月度合规运营：¥10,000-50,000
```

## AI 生成内容标识

根据《人工智能生成合成内容标识办法》，生成合成内容标识包括显式标识和隐式标识，办法已于 2025-09-01 施行。

产品应做：

- 导出页提示：本视频含 AI 生成 / 合成内容。
- 默认显式水印或结尾标识。
- 元数据中写入 AI 生成标识。
- 发布前提示用户按平台规则声明 AI 内容。
- 去水印不等于取消 AI 标识义务。

## 肖像与声音授权

数字人和声音克隆涉及高风险个人信息。

必须做：

- 上传人像时确认本人授权。
- 上传声音时确认本人授权。
- 禁止上传明星、网红、他人照片或声音进行冒充。
- 保存授权记录。
- 高级能力可要求手机号实名 / 企业认证 / 活体确认。
- 用户协议中明确素材权利归属和侵权责任。

## 用户素材和数据出境

如果使用 OpenAI、fal、RunPod 等境外 API 处理用户素材，可能涉及个人信息出境，尤其是人脸、声音、门店视频、客户评价截图。

建议策略：

- GPT-5.5 尽量只处理脱敏文本。
- 真人脸和声音尽量走境内合规供应商。
- 如使用境外 API，要在隐私政策中明确告知。
- 对敏感个人信息或出境处理取得单独同意。
- 用户素材默认不用于训练。
- 明确素材保存期限，例如 30 / 90 / 180 天。
- 提供删除入口。

## 电商广告合规

生成的视频很多会成为广告或营销素材。

高风险行业：

- 医疗。
- 药品。
- 保健品。
- 医美。
- 金融。
- 教育培训。
- 招商加盟。
- 高收益承诺。
- 成人、灰产、违规引流。

需要拦截或提示的词：

- 根治。
- 治愈。
- 最强。
- 第一。
- 100% 有效。
- 包过。
- 稳赚。
- 无风险。
- 国家级、最高级等绝对化表述。

产品应做：

```text
脚本生成前行业识别
高风险词拦截
生成结果发布前确认
商品价格、优惠、功效、证书由用户确认真实性
保留生成和确认记录
```

## ICP 和支付

如果服务部署在中国大陆服务器：

- 通常需要 ICP 备案。
- 可能需要公安联网备案。
- 收费 SaaS 是否需要 ICP 许可证，要按业务模式咨询律师或当地通信管理部门。

早期可用香港 / 新加坡服务器快速验证，但如果主做中国大陆商家，长期建议：

```text
大陆主体
ICP备案
公安联网备案
微信 / 支付宝商户
必要时确认 ICP 许可证
```

## 税务要点

早期常见主体：有限公司。

### 小规模纳税人

如果符合小规模纳税人条件：

- 年应税销售额 500 万元及以下。
- 月销售额 10 万元以下可能享受增值税免征。
- 适用 3% 征收率的应税收入，目前有减按 1% 征收政策，延续至 2027-12-31。

### 小型微利企业所得税

在满足条件时，企业所得税实际税负可能约 5%：

```text
应纳税所得额 × 25% × 20%
```

该优惠政策当前延续至 2027-12-31。

### 一般纳税人

如果年销售额超过 500 万，可能转一般纳税人。

SaaS / 信息技术服务通常按现代服务 / 信息技术服务方向处理，增值税税率可能是 6%，但可抵扣进项税。

风险：

```text
境外 API / 云 GPU 费用如果没有合规凭证或发票，未来抵扣和税前扣除会变麻烦。
```

### 分红个税

公司税后利润如果分红给个人，股息红利个人所得税通常为 20%。

所以：

```text
公司税后净利润 ≠ 老板个人到手。
```

## 风控建议

第一版就要做：

- 素材上传承诺。
- 肖像 / 声音授权勾选。
- AI 内容标识。
- 高风险行业拦截。
- 敏感词审核。
- 用户协议 / 隐私政策 / 付费协议。
- 生成失败补偿规则。
- 退款规则。
- 任务日志和生成记录。

后续再做：

- 企业认证。
- 人脸活体验证。
- 自动版权检测。
- 更复杂的内容审核 API。
- 法务工单系统。
