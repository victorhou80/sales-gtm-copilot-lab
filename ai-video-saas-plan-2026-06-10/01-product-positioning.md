# 产品定位

## 推荐定位

产品定位为：

```text
给小店主、小电商、小型博主使用的手机 AI 视频生成 SaaS。
```

不是：

```text
本地 GPU 视频生成工具
代运营服务
纯 AI 电影级视频生成器
复杂剪辑软件
```

用户真正需要的是：

```text
上传几张商品图 / 店铺图
输入几句卖点
选择模板
确认脚本
几分钟到十几分钟后拿到 1080p30 成片
下载后发抖音、小红书、视频号、朋友圈
```

## 目标用户

### 小店主

典型用户：

- 水果店、餐饮店、奶茶店、花店。
- 美甲、美容、美发、宠物店。
- 本地生活小商家。
- 个体店主、夫妻店、社区店。

核心需求：

- 店铺介绍。
- 新品介绍。
- 促销活动。
- 老板口播。
- 商品图变短视频。
- 不会剪辑也能发短视频。

### 小电商

典型用户：

- 淘宝、拼多多、抖店、视频号小店、闲鱼卖家。
- 多 SKU 小卖家。
- 私域团购、小批量货盘商。

核心需求：

- 批量商品介绍视频。
- 商品卖点自动提取。
- 产品图片变视频。
- 多平台比例导出。
- 标题、封面、口播文案。

### 小型博主

典型用户：

- 知识博主。
- 财经 / 历史 / 人文 / 生活方式博主。
- 图文转视频创作者。
- 没有剪辑团队的个人账号。

核心需求：

- 文章 / 主题转解说脚本。
- 配图、字幕、旁白。
- 封面标题。
- 1-5 分钟图文解说视频。

## 核心视频类型

### 1. 轻量图文视频

结构：

```text
商品图 / 店铺图 / 素材视频
+ GPT-5.5 脚本
+ TTS 配音
+ 字幕
+ 模板动效
+ FFmpeg / Remotion 合成
```

特点：

- 成本最低。
- 生成最快。
- 适合普通套餐大量提供。
- 适合电商批量 SKU。

### 2. 单人口播 / 数字人视频

结构：

```text
人像 / 数字人
+ TTS 或用户录音
+ InfiniteTalk / 其他数字人模型
+ 商品图和字幕叠加
```

特点：

- 对小店主很有吸引力。
- 能做「老板介绍店铺」「小博主讲解」。
- 必须按分钟限额。
- 不适合无限送。

### 3. AI 高级镜头

结构：

```text
商品图 / 故事板图
+ WAN 或 Seedance I2V
+ 5-15 秒短镜头
+ 后期拼接
```

特点：

- 能显著提升「AI 质感」。
- 成本高。
- 失败重抽成本高。
- 应该按秒 / 点数收费。

### 4. 5 分钟以内长一点的解说视频

结构：

```text
主题 / 文案 / 商品资料
+ GPT-5.5 长脚本
+ TTS
+ 多章节字幕
+ 图片 / 图表 / 商品素材
+ 少量 AI 镜头
```

特点：

- 适合小博主、知识号。
- 不应该全程 AI 视频生成。
- 主要靠脚本、配音、图文、字幕和模板完成。

## 不建议第一版做的东西

- 不做全片纯 AI 生成 5 分钟视频。
- 不做复杂本地安装工具。
- 不让客户电脑承担 GPU 计算。
- 不做自动发布到抖音 / 小红书。
- 不做代运营。
- 不承诺帮客户涨粉。
- 不做无限高级视频套餐。
